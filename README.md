# VeganScan

AI-powered vegan menu analyzer

## Overview

Users input restaurant menu URLs and get vegan suitability analysis using LLM.

## Why

Built to solve real-world vegan dining friction.

## Features

- URL input
- Vegan classification
- Explanation
- User analytics integration

## Tech

- React
- Express.js
- Docker
- AWS EC2, DynamoDB
- LLM API
- GA4
- BigQuery
